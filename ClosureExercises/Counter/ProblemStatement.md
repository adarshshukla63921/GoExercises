# Exercise: Stateful Counter Using Closures

## Difficulty
Easy

## Time Limit
30 minutes

## Concepts Covered
- Structs
- Pointers
- Closures
- Encapsulation of state

---

## Problem Statement

You are required to implement a **stateful counter** using **closures** in Go.

The counter should internally store its state using a **struct**, and that state must be:
- **Private**
- **Persisted across function calls**
- **Accessible only through retur**
